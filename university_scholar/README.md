# University Scholar Module

Module Odoo pour gérer les enseignants et leurs publications scientifiques via Google Scholar (SerpAPI).

## Installation

1. Placez le module dans le répertoire `addons` d'Odoo
2. Mettez à jour la liste des applications dans Odoo
3. Installez le module "University Scholar"

## Configuration

1. Obtenez une clé API sur https://serpapi.com/
2. Allez dans Paramètres > University Scholar
3. Entrez votre clé API SerpAPI
4. Sauvegardez

## Utilisation

1. Créez un enseignant dans Université > Enseignants
2. Renseignez le Google Scholar ID
3. Cliquez sur "Synchroniser avec Google Scholar"
4. Les articles seront importés automatiquement
